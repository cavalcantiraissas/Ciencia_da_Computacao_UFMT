# 🖥️ Programação de Computadores

Bem-vindo ao repositório da disciplina **Programação de Computadores**! 🚀 Aqui você encontrará materiais, exercícios e resumos para auxiliar no aprendizado da programação estruturada em Python.

## 🎯 Objetivo Geral
Introduzir o aluno à **lógica de programação**, solução algorítmica de problemas e **programação estruturada** em Python.

## 📚 Conteúdos Abordados
- 🧩 **Conceitos Fundamentais** de Programação
- 🔄 **Estruturas Básicas**: Condicionais, Laços e Funções
- 📦 **Estruturas de Dados Simples**: Listas, Tuplas, Dicionários
- ⚙️ **Modularização** e Reutilização de Código
- 📂 **Manipulação de Arquivos e Memória**
- 💻 **Programação Prática** e Resolução de Problemas

## 🔧 Tecnologias e Ferramentas
- **Linguagem**: Python 🐍
- **Ferramentas**: Jupyter Notebook, VS Code, Git
