# 🖥️ Programação II 

Bem-vindo ao repositório da disciplina **Programação II**! 🚀 Aqui você encontrará materiais, exercícios e resumos para auxiliar no aprendizado de **C++** e seus conceitos avançados.

## 🎯 Objetivo Geral
Aprofundar os conhecimentos em **programação estruturada e orientada a objetos (POO)** utilizando **C++**, abordando boas práticas e técnicas modernas de desenvolvimento.

## 📚 Conteúdos Abordados
- 🖥️ **Introdução ao C++** e diferenças em relação ao C
- 🔄 **Controle de Fluxo**: Estruturas condicionais e loops
- 🧠 **Gerenciamento Dinâmico de Memória**: Alocação e desalocação
- 🏗️ **Programação Orientada a Objetos (POO)**
  - Classes e Objetos
  - Encapsulamento, Herança e Polimorfismo
- 📦 **Utilização de Classes Pré-definidas**
- ➕ **Sobrecarga de Operadores**
- ⚠️ **Tratamento de Exceções**
- 🔧 **Programação Genérica**: Templates e STL (Standard Template Library)

## 🔧 Tecnologias e Ferramentas
- **Linguagem**: C++ 🔹
- **Compiladores**: GCC, Clang, MSVC
- **Ambientes de Desenvolvimento**: VS Code, CLion, Code::Blocks
- **Versionamento**: Git & GitHub
