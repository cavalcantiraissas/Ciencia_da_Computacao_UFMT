# 📂 Estrutura de Dados I 

Bem-vindo ao repositório da disciplina **Estrutura de Dados I**! 🚀 Aqui você encontrará materiais, exercícios e resumos para auxiliar no aprendizado da implementação e análise de estruturas de dados utilizando **C**.

## 🎯 Objetivo Geral
Compreender e implementar **estruturas de dados** fundamentais, analisando seu desempenho e aplicabilidade na solução de problemas computacionais.

## 📚 Conteúdos Abordados
- 🖥️ **Introdução à Linguagem C**
- 📑 **Estruturas de Dados Lineares**
  - Listas, Pilhas e Filas
- 🔢 **Algoritmos de Ordenação**
- 📊 **Introdução à Análise de Algoritmos**
- 🌳 **Estruturas Não Lineares**
  - Árvores, Grafos e Heap

## 🔧 Tecnologias e Ferramentas
- **Linguagem**: C 🖥️
- **Ambientes de Desenvolvimento**: GCC, VS Code, CLion
- **Versionamento**: Git & GitHub
