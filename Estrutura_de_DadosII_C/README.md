# 🌳 Estrutura de Dados IIc

Bem-vindo ao repositório da disciplina **Estrutura de Dados II**! 🚀 Aqui você encontrará materiais, exercícios e resumos para aprofundar seus conhecimentos em **estruturas avançadas de dados** utilizando **C**.

## 🎯 Objetivo Geral
Compreender, implementar e analisar **estruturas de dados avançadas**, com foco em eficiência e aplicações práticas, incluindo árvores balanceadas e grafos.

## 📚 Conteúdos Abordados
- 📂 **Estruturas Avançadas de Dados**
  - Listas e Filas Avançadas
  - Hashing e Tabelas Hash
- 🌳 **Árvores Balanceadas**
  - Árvores AVL
  - Árvores Rubro-Negras
  - Árvores B e B+
- 🔗 **Grafos e Aplicações**
  - Representações de Grafos (Matriz de Adjacência, Lista de Adjacência)
  - Algoritmos de Caminhamento (BFS e DFS)
  - Algoritmos de Caminho Mínimo (Dijkstra, Bellman-Ford, Floyd-Warshall)
  - Árvore Geradora Mínima (Kruskal, Prim)
  - Fluxo Máximo

## 🔧 Tecnologias e Ferramentas
- **Linguagem**: C 🖥️
- **Compiladores**: GCC, Clang
- **Ambientes de Desenvolvimento**: VS Code, CLion
- **Versionamento**: Git & GitHub
